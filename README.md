# hbdeeplearningjari

Download atau install terlebih dahulu OpenCV dan MediaPipe pada terminal :

pip install opencv-python

pip install mediapipe==0.10.14

MediaPipe versi 0.10.14 dipilih karena merupakan salah satu versi yang stabil dan kompatibel untuk implementasi deteksi tangan menggunakan modul mp.solutions.hands. Versi ini masih mendukung penggunaan API MediaPipe Solutions yang banyak digunakan dalam berbagai contoh implementasi Computer Vision berbasis Python. Selain itu, pada beberapa versi MediaPipe yang lebih baru, struktur paket dan API mengalami perubahan sehingga dapat menyebabkan error.
